# Essential lints

This is the root for the `essential_lints` package and `essential_lints_annotations` package, which provides additional lint rules for Dart and Flutter projects.

## Packages

- [essential_lints](https://pub.dev/packages/essential_lints): A collection of new lint rules for Dart/Flutter projects and related fixes.
- [essential_lints_annotations](https://pub.dev/packages/essential_lints_annotations): Annotations to configure lint rules from the `essential_lints` package.
